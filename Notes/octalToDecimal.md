## Octal to Decimal
---
Octal allows 8 digits 0-7

Convert decimal (base 10) to octal (base 8):

1. modulus
2. division (integer division)
Termination condition: division returns 0
```
15_10 == ?_8
--------------
15%8 = 7
15/8 = 1
--------------
1 % 8 = 1
1 / 8 = 0
-------------
17_8

```
